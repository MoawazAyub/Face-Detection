Face Detection

Face Dection is a project created in MATLAB for facial recognition.

How to get started:

First run gettingpictures.m in MATLAB editor. This script will turn on the camera of your computer and take your pictures.
After that run resizing.m to resize pictures.
After resizing run real.m that file will make the training data and test data and try to identify the images in the testing data.
